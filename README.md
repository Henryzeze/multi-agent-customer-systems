# multi-agent-customer-systems
Multi-Agent Customer Systems for Customer support automation, and Customer outreach campaign

## 📌 Overview
This project implements a **multi-agent agentic system** designed to perform Customer support automation and Customer outreach campaigns.

The system leverages **LLM-powered agents** to collaboratively perform tasks such as planning, reasoning, and execution in a coordinated workflow.

---

## 🏗️ System Architecture

The system consists of multiple specialized agents and key elements that help make Agents perform even better :

- Role Playing
- Focus
- Tools
- Cooperation
- Guardrails
- Memory
- Versatility
- Fault Tolerance
- Caching

---

## 🧰 Technologies Used

- Python  
- CrewAI  
- LangChain  
- OpenAI API / LLMs
